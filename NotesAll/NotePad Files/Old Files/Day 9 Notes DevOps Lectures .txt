Some TroubleShooting Commands /Tools in Linuz To check the server:

1. netstat -lntp --> network statistics 
2. ps -ef | grep <servername> --> Gives the Server name with port number and status of port 
3. systemctl status servername --> gives status of the server 
4. top --> Gives the cpu process details along with server details
